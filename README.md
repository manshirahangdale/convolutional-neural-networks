## Observations
- CNNs significantly outperform ANN for image data because spatial information is preserved.
- Increasing model depth improves feature extraction but also increases overfitting risk.
- Pooling layers help reduce computation but may discard fine-grained details.
- Proper normalization and regularization are important for stable training.
